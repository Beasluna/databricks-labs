## 🧠 Core Concepts | Conceptos Clave

### Batch vs. Streaming
* **Batch Processing:** Analyzing large static datasets (Titanic, WorldHits).
* **Streaming Processing:** Real-time event handling using Kafka and Spark.

### Lakehouse Architecture
The curriculum focuses on the evolution from traditional Data Lakes to **Delta Lake**, enabling ACID transactions and time-travel features.

> **[ES]** El programa se centra en la evolución de los Data Lakes tradicionales hacia **Delta Lake**, permitiendo transacciones ACID y funciones de *time-travel*.
